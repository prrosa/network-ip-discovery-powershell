# Network IP Discovery - PowerShell

Script em PowerShell para identificar IPs ativos em uma rede local (ping sweep).

## Objetivo
Automatizar descoberta de hosts ativos, útil para inventário de ativos, troubleshooting e auditoria de rede.

## Uso
Abra PowerShell, navegue até o repositório e execute:

```powershell
.\ip_scan.ps1
